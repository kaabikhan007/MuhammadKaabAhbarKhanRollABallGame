# MuhammadKaabAhbarKhanRollABallGame
Mid-Term Assingment

🎮 My Unity Game Development Diary:

Step 1: Setting up the project 🏗️

Created a new Unity project and organized folders for Scenes, Scripts, and Assets.
Imported initial assets and set up the main scene.
Learned to navigate Unity’s interface and manage the hierarchy.
Fixed missing script references by reassigning them in the Inspector.

Step 2: Moving the player 🕹️

Implemented player movement using Rigidbody and Input controls.
Adjusted speed and physics for smooth movement.
Learned about FixedUpdate vs Update.
Fixed clipping issues by adjusting colliders.

Step 3: Camera control 🎥

Added a camera that follows the player smoothly.
Explored Cinemachine for advanced camera control.
Tweaked offset and damping for a polished gameplay experience.

Step 4: Setting up the play area 🌳

Designed the playable area with boundaries and static obstacles.
Used Mesh Colliders or Box Colliders depending on obstacle shapes.
Baked a NavMesh for AI navigation.

Step 5: Creating collectibles ✨

Added collectible items in the play area.
Used triggers and colliders for detection.
Added a UI canvas to show player score dynamically.

Step 6: Detecting collisions and scoring 🏆
Increased player score when collecting items.
Updated UI text in real-time via C# scripts.
Debugged collision issues using Unity Console.

Step 7: Adding AI navigation 🤖

Added AI enemies using NavMeshAgents.
Fixed enemies passing through obstacles by adding proper colliders.
Implemented dynamic obstacles (moving/rotating) for AI interaction.

Step 8: Bonus work – assets & polish 🎨

Installed additional Unity packages for extra functionality.
Added more environment assets and a background skybox.
Fixed errors and improved gameplay stability.
Created dynamic obstacles to make the game more challenging.

Step 9: Building the game 🖥️

Configured Build Settings for Windows and macOS.
Added all scenes to the build.
Verified player settings, resolution, and icons.
Created a standalone build and tested gameplay successfully.

Reflections 💡

Learned about player movement, camera control, collisions, AI navigation, and dynamic obstacles.
Troubleshooting taught me the importance of colliders, NavMesh, and testing frequently.
Using AI (ChatGPT) helped me solve collider issues, build errors, and write commit messages.
Loved seeing all elements come together in a playable game.
