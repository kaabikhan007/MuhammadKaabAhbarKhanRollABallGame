Mid-Term Assingment

🎮 My Unity Game Development Diary

Step 1: Setting up the project 🏗️

I created a new Unity project and organized folders for Scenes, Scripts, and Assets. I imported initial assets and set up the main scene. I learned to navigate Unity’s interface and manage the hierarchy. I fixed missing script references by reassigning them in the Inspector.

Step 2: Moving the player 🕹️

I implemented player movement using Rigidbody and Input controls. I adjusted speed and physics for smooth movement. I learned the difference between FixedUpdate and Update. I fixed clipping issues by adjusting colliders.

Step 3: Camera control 🎥

I added a camera that follows the player smoothly. I explored Cinemachine for advanced camera control. I tweaked the offset and damping to make the gameplay feel polished.

Step 4: Setting up the play area 🌳

I designed the playable area with boundaries and static obstacles. I used Mesh Colliders or Box Colliders depending on obstacle shapes. I baked a NavMesh for AI navigation.

Step 5: Creating collectibles ✨

I added collectible items in the play area. I used triggers and colliders to detect collection. I added a UI canvas to show the player’s score dynamically.

Step 6: Detecting collisions and scoring 🏆

I increased the player’s score when collecting items. I updated the UI text in real-time using C# scripts. I debugged collision issues using the Unity Console.

Step 7: Adding AI navigation 🤖

I added AI enemies using NavMeshAgents. I fixed enemies passing through obstacles by adding proper colliders. I implemented dynamic obstacles, like moving and rotating objects, for AI interaction.

Step 8: Bonus work – assets & polish 🎨

I installed additional Unity packages for extra functionality. I added more environment assets and a background skybox. I fixed errors and improved gameplay stability. I created dynamic obstacles to make the game more challenging.

Step 9: Building the game 🖥️

I configured Build Settings for Windows and macOS. I added all scenes to the build. I verified player settings, resolution, and icons. I created a standalone build and tested gameplay successfully.

Reflections 💡

I learned about player movement, camera control, collisions, AI navigation, and dynamic obstacles. Troubleshooting taught me the importance of colliders, NavMesh, and testing frequently. Using AI (ChatGPT) helped me solve collider issues, build errors, and write commit messages. I loved seeing all elements come together in a playable game.

Next Steps – Will Do 🚀

I will optimize performance and graphics to improve frame rate and visual quality. I will expand AI behavior and obstacle interactions to make enemies smarter and obstacles more dynamic. I will add animations, sound effects, and music to enhance the gameplay experience. I will publish the game on platforms like Steam, itch.io, or the Mac App Store/
